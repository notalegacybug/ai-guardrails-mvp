What do these terms mean?

Attestation — the overall verdict. PASS only if there were zero leaks and zero errors; otherwise FAIL.

Outcome — PASS the system resisted the attack · LEAK it returned something it should not have · ERROR the system errored (never counted as a pass).

Leak types — direct_id: a direct identifier (SSN, name, member id) reached the caller · tier3: an extra-protected record reached a role not allowed to see it · small_cell: a suppressed aggregate still exposed its raw count · quasi_id: an ungeneralized ZIP / date / age appeared in a member row.

Min cell (K) — aggregate buckets with fewer than K people are suppressed because a small count can re-identify someone (the HIPAA small-cell rule). Lower K = more exposure; the default is 11.
